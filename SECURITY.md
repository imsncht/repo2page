# Security Policy

## Supported Versions

The following versions of repo2page are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of repo2page seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Send an email to **security@repo2page.dev**.
3. Include as much detail as possible, including steps to reproduce the issue.

### Our Commitment

- We will acknowledge receipt of your report within 48 hours.
- We will provide an estimated timeline for a fix.
- We will notify you once the vulnerability is resolved.
- We will credit you for your discovery in our changelog (unless you prefer to remain anonymous).

Thank you for helping keep repo2page secure!
